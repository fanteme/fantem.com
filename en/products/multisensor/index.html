OOMI Six-in-One MultiSensor

OOMI Multisensor

Increase your home security and get notified if something unexpected happens. MultiSensor can also trigger customized automations. Have lights illuminate as it gets dark or turn off when you’re not around.

OOMI Multisensor智能六合一传感器是一款多功能的传感器设备，集温度、湿度、亮度、紫外线、震动、红外移动监测于一体。

Proper humidity provides comfortable home.

Through presetting, when extreme humidity is detected, MultiSenor can send instructions to linked device(s) to adjust the indoor humidity.

湿度传感器 缔造舒适环境

当传感器检测到空气过于干燥时，传感器向网关发送指令，通过联动OOMI智能插座可以打开与其相连的加湿器，改善空气湿度。

Pleasant temperature makes home adorable.

Through presetting, when extreme temperature is detected, MultiSenor can send instructions to linked device(s) to adjust the indoor temperature.

Light sensor controls indoor lighting.

智能光控感应，总是为您的舒适着想。

By presetting light threshold, MultiSensor can detect indoor lightness and makes indoor lights dimming intelligently to catch the preset light threshold.

Vibration sensor secures safety.

When abnormal vibration is detected, MultiSensor can report to Cube and notify user by sending message to mobile phone.

Motion sensor knows something is moving.

MultiSensor can detect movement and allow user to check related message on mobile phone.

UV sensor cares most about the home.

With high sensitivity UV sensor, MultiSensor can efficiently detect indoor UV intensity and record it in real time.
