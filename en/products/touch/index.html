OOMI Touch

The whole home controller

OOMI Touch is the whole home controller of the OOMI Smart Home System. By using it user can master the entire home.

小身板，大功能，方寸之间蕴含多种可能

OOMI Touch provides programmable tactile buttons and customizable widgets, allowing user to control and monitor the entire home through its 7’’ touch screen as well as get access to tutorials and configure the smart home system.

NFC

With a tap of OOMI Touch can user instantaneously connect OOMI devices to the smart home system. This is Tap-and-Touch, OOMI’s revolutionary setup technology that based on NFC, making installation incredibly easy.

Customized Automation

OOMI Touch supports user to set up scene or automation based on personal habit or preference so that user can enjoy the convenient and intelligent life brought by OOMI Smart Home.

Consumption Monitoring

User can check the energy consumption of specific device or the entire home on the OOMI Touch in real time.

Surveillance Video Checking

User can slid to the monitoring page and check the surveillance video on the OOMI Touch, having the overall view of the entire house.
